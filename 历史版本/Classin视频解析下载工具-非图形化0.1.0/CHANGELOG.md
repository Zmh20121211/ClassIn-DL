# ClassIn视频下载工具 - 非图形化版本更新日志

## [0.1.0] - 2023-01-01

### 新增功能

- 初始版本发布
- 基本的视频解析功能：可以解析ClassIn平台的视频链接
- 简单的下载功能：支持视频下载
- 命令行操作：使用批处理文件执行所有操作

### 使用方法

1. 打开`classin网址解析.bat`文件
2. 输入`1`并按回车开始解析
3. 工具会自动从剪贴板中解析视频链接和名称
4. 解析成功后，会显示解析结果
5. 可以重复解析多个视频
6. 输入`2`并按回车结束解析
7. 按回车开始下载视频
8. 下载完成后，视频会保存在当前目录的`下载目录`文件夹中

### 注意事项

- 本工具仅用于个人学习和备份，请勿用于商业用途
- 请遵守ClassIn平台的使用协议
- 下载过程中请保持网络连接稳定
- 工具会在当前目录生成临时文件`请求头缓存.dll`，使用完成后可手动删除
- 工具会生成`解析结果.ini`文件，记录解析成功的视频信息
- 工具会生成`下载视频工具.bat`文件，用于执行下载操作
