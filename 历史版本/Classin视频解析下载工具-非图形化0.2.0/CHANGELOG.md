# ClassIn视频下载工具 - 非图形化版本更新日志

## [0.2.0] - 2023-02-01

### 新增功能

- 增加了aria.exe下载支持：使用aria下载器，提升下载速度
- 支持多线程下载：优化了下载速度
- 改进了解析算法：更准确地解析视频链接
- 增加了错误处理：更好地处理解析失败的情况

### 改进功能

- 优化了用户交互：更清晰的提示信息
- 增强了稳定性：修复了已知问题

### 使用方法变更

1. 打开`classin网址解析.bat`文件
2. 输入`1`并按回车开始解析
3. 工具会自动从剪贴板中解析视频链接和名称
4. 解析成功后，会显示解析结果
5. 可以重复解析多个视频
6. 输入`2`并按回车结束解析
7. 按回车开始下载视频（工具会自动调用aria.exe进行下载）
8. 下载完成后，视频会保存在当前目录的`下载目录`文件夹中

### 注意事项

- 确保当前目录中有`aria.exe`文件
- 本工具仅用于个人学习和备份，请勿用于商业用途
- 请遵守ClassIn平台的使用协议
- 下载过程中请保持网络连接稳定
