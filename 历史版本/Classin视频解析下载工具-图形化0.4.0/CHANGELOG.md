# ClassIn视频下载工具 - 图形化版本更新日志

## [0.4.0] - 2023-04-01

### 新增功能

- 初始图形化版本发布
- 基本的图形化界面：提供直观的操作界面
- 视频解析功能：可以解析ClassIn平台的视频链接
- 多线程下载：支持多个视频同时下载
- 下载进度显示：可以查看每个视频的下载进度
- 下载信息显示：显示已下载大小、总大小、当前速度和剩余时间
- 单个视频下载：支持单独下载某个视频
- 打开下载目录：方便查看下载结果
- 更改下载路径：支持自定义视频保存位置

### 使用方法

1. 运行`Classin视频解析下载工具.exe`
2. （可选）点击`复制命令`按钮，将`getLessonRecordInfo`复制到剪贴板
3. 点击`解析`按钮，工具会自动从剪贴板中解析视频链接和名称
4. 解析成功后，视频信息会显示在列表中
5. （可选）点击`更改路径`按钮，选择视频保存位置
6. （可选）调整`最大并发下载数`和`最大下载线程数`
7. 点击`开始下载`按钮，开始下载视频
8. 下载过程中可以查看每个视频的下载进度
9. 下载完成后，视频会保存在指定的下载目录中
10. 可以点击`打开文件夹`按钮直接打开下载目录

### 注意事项

- 确保已经正确复制了包含`getLessonRecordInfo`的请求头
- 确保HTTP Debugger Pro捕获到了完整的请求
- 本工具仅用于个人学习和备份，请勿用于商业用途
- 请遵守ClassIn平台的使用协议
